# Расчёт суточной нормы кормления смесью у детей 0–6 месяцев
## Канонический консолидированный спек (финальная редакция)

**Назначение.** Технический документ для команды разработки: формулы, константы, границы валидации и алгоритм расчёта суточного объёма молочной смеси для детей на искусственном вскармливании 0–6 месяцев. Источники — только авторитетные международные (WHO/FAO/UNU, AAP, ESPGHAN, Codex Alimentarius, NICE, CDC, EFSA, IOM).

**Провенанс.** Документ — итог нескольких раундов перекрёстной сверки двух независимых исследований (Claude + GPT). Вошли только позиции, на которых стороны сошлись или которые подтверждены по первоисточникам. Спорное разрешено, ошибки исключены (раздел 9).

## 1. Краткая сводка для имплементации

1. **Основной режим — энергетический (FAO/WHO/UNU):** учитывает возрастное снижение ккал/кг. AAP-метод показывать рядом как sanity-check. ✅
2. **Таблица FAO для искусственников (ккал/кг/сут):** 1 мес — 120, 2 мес — 109, 3 мес — 100, 4 мес — 87, 5 мес — 86, 6 мес — 84. ✅
3. **Перевод:** `V_ml_day = E_kcal_day × 100 / D`, где `D` — ккал/100 мл смеси. ✅
4. **Плотность `D`:** с этикетки; fallback `67` (помечать «estimate»); коридор стандартной смеси Codex 60–70. ✅
5. **AAP массо-объёмный:** `V = масса_кг × 165`, мягкий потолок ~960 мл/сут (источник — **только AAP**). ✅/🔧
6. **Первая ~неделя–две** — per-feed + кратность, не единая суточная норма; точный порог = product policy. 🏷️
7. **Responsive feeding** — обязательный disclaimer: расчёт = ориентир, не цель. ✅

## 2. Основной режим: энергетический метод (FAO/WHO/UNU 2004) ✅

### 2.1. Формула
```
E_kcal_day = масса_кг × kcal_per_kg(возраст_мес)
V_ml_day   = E_kcal_day × 100 / D_kcal_per_100ml
```

### 2.2. Таблица потребности (formula-fed), FAO/WHO/UNU 2004, Table 3.3

| Возраст (мес) | ккал/кг/сут | мл/кг/сут при D=67 | мл/кг/сут при D=67,63 |
|---:|---:|---:|---:|
| 1 | 120 | 179,1 | 177,4 |
| 2 | 109 | 162,7 | 161,2 |
| 3 | 100 | 149,3 | 147,9 |
| 4 | 87 | 129,9 | 128,6 |
| 5 | 86 | 128,4 | 127,2 |
| 6 | 84 | 125,4 | 124,2 |

Значения сошлись независимо в обоих исследованиях и сверены с первоисточником. FAO даёт отдельные колонки breastfed / formula-fed / all infants; это колонка для искусственного вскармливания (у искусственников TEE выше: +12% в 3 мес, +7% в 6 мес).
Источник: FAO, *Human energy requirements* (2004), гл. 3 — `https://www.fao.org/4/y5686e/y5686e05.htm`

### 2.3. Опорные медианные веса (WHO 1994, использованы в FAO 2004)
Только если фактический вес неизвестен (предпочтительно требовать реальный вес).

| Возраст (мес) | Мальчики | Девочки | Среднее |
|---:|---:|---:|---:|
| 1 | 4,58 | 4,35 | 4,47 |
| 2 | 5,50 | 5,14 | 5,32 |
| 3 | 6,28 | 5,82 | 6,05 |
| 4 | 6,94 | 6,41 | 6,68 |
| 5 | 7,48 | 6,92 | 7,20 |
| 6 | 7,93 | 7,35 | 7,64 |

### 2.4. Регрессионные уравнения TEE (FAO/WHO/UNU 2004) — расширенный режим ✅
Сверены дословно по первоисточнику. Дают TEE без энергии роста (её прибавляют отдельно, см. 2.5).
```
Все младенцы:        TEE_kcal_day = -99,4  + 88,6 × масса_кг
Искусственники:      TEE_kcal_day = -29,0  + 82,6 × масса_кг
Грудное вскармл.:    TEE_kcal_day = -152,0 + 92,8 × масса_кг
```
Для рядового калькулятора достаточно табличного метода 2.2.

### 2.5. Альтернатива — IOM 2002 (простая форма) ✅
```
EER_kcal_day = (89 × масса_кг − 100) + энергия_роста
энергия_роста: 175 ккал/сут (0–3 мес); 56 ккал/сут (4–6 мес)
```
Источник: IOM 2002 (воспроизведено в AAP Pediatric Nutrition Handbook).

## 3. Sanity-check режим: массо-объёмный метод (AAP) ✅/🔧

### 3.1. Формула
```
V_ml_day = масса_кг × 165          // округлённое «75 мл на 453 г»
// точная конвенция-альтернатива: × 163,0 (2,5 oz/lb)
V_ml_day = min(V_ml_day, 960)      // мягкий ориентир, §3.2
```
AAP: в среднем ~75 мл (2,5 oz) смеси в сутки на каждые 453 г массы.
Источник: AAP / HealthyChildren, *Amount and Schedule of Baby Formula Feedings* (2022) — `https://www.healthychildren.org/English/ages-stages/baby/formula-feeding/Pages/amount-and-schedule-of-formula-feedings.aspx`

### 3.2. 🔧 Атрибуция и характер потолка 960 мл
- Потолок ~960 мл/сут (32 oz) — **только AAP**, не «AAP/CDC». Проверено: страница CDC по кормлению суточного лимита не содержит; 32 oz у CDC относится к достаточности витамина D (≥32 oz → добавка не нужна) — это порог-снизу, не потолок-сверху.
- Формулировка AAP мягкая («usually no more than»), поэтому в коде — **soft guardrail / флаг**, severity = `warning`, source = `AAP`. Это практический порог перекорма, не токсикологический предел.

## 4. Обработка плотности смеси `D` ✅
```
if label_kcal_per_100ml: D = label_kcal_per_100ml
elif label_kcal_per_oz:  D = label_kcal_per_oz / 29,5735 × 100
else:                    D = 67;  flag "estimated_density_default"
```

| Этикетка | ккал/100 мл | Контекст |
|---|---:|---|
| 19 ккал/oz | 64,25 | term formula (нижняя типичная) |
| 20 ккал/oz | 67,63 | самый типичный стандарт |
| ~67 (default) | 67,00 | рабочее значение WHO/JECFA, fallback |
| 22 ккал/oz | 74,39 | post-discharge / enriched |
| 24 ккал/oz | 81,15 | preterm / fortified |

Codex STAN 72-1981: стандартная infant formula — 60–70 ккал/100 мл. `D` вне 60–70 при `standard_formula = true` → предупреждение.
Источник: Codex Alimentarius CXS 72-1981 — `https://www.fao.org/input/download/standards/288/CXS_072e.pdf`

## 5. Возрастное ветвление

### 5.1. 🏷️ Ранний период (первые ~1–2 недели) — PRODUCT POLICY
Единая суточная норма НЕ выдаётся; показывать объём на кормление + кратность + responsive feeding.

**Важно (правка кросс-сверки):** точный порог переключения — это **продуктовое решение**, а не клиническая константа. CDC и AAP покрывают «первые дни» и «первые недели» **без резкой границы на дне 14**. Поэтому:
```
EARLY_NEWBORN_DAYS = 14    // configurable, status = product_policy
if age_days < EARLY_NEWBORN_DAYS: return early_newborn_guidance
```
Содержание guidance: 30–60 мл (1–2 oz) на кормление, 8–12 кормлений/сут, кормление по сигналам голода/сытости.
Источник: CDC, *How Much and How Often to Feed Infant Formula* — `https://www.cdc.gov/infant-toddler-nutrition/formula-feeding/how-much-and-how-often.html`

### 5.2. Доношенный 1–2 недели – 6 месяцев ✅
Основной режим — FAO energy (§2), рядом AAP-simple (§3) как проверка. Бакет возраста — по завершённым месяцам; правило округления дробных значений зафиксировать и задокументировать.

## 6. Поведение двух методов по возрасту (важно для UX) ✅
AAP-simple — плоские ~165 мл/кг; FAO — снижающиеся ккал/кг. Расхождение неравномерно (медианные веса, D=67):

| Возраст | AAP-simple, мл/сут | FAO energy, мл/сут | Комментарий |
|---|---:|---:|---|
| 1 мес (4,5 кг) | 743 | 806 | FAO выше ~8% |
| 2 мес (5,3 кг) | 875 | 862 | практически совпадают |
| 3 мес (6,1 кг) | 1007 → cap 960 | 910 | AAP без капа уходит вверх |
| 4 мес (6,7 кг) | 1106 → cap 960 | 870 | расхождение максимально |
| 5 мес (7,2 кг) | 1188 → cap 960 | 924 | сходятся к капу |
| 6 мес (7,6 кг) | 1254 → cap 960 | 953 | оба ≈ 950–960 |

**Следствия для кода:**
- С ~3–4 мес плоское AAP завышает; потолок 960 возвращает его к FAO-уровню → аргумент за «FAO основной».
- Расхождение методов на 8–25% (особенно ~1 и ~4 мес) — **ожидаемо**, не трактовать как ошибку ввода.
- Если в UI нужно одно «головное» число — брать FAO (физиологичнее и не зависит от того, спас ли cap). AAP показывать вторым.

## 7. Параметры валидации и триггеры

### 7.1. Жёсткие проверки ввода (HARD) ✅
- `масса_кг > 0`; `возраст_дни ≥ 0`
- если заданы и kcal/100ml, и kcal/oz и расходятся > 2% → ошибка
- `D` вне 60–70 при `standard_formula = true` → предупреждение

### 7.2. Ориентировочные диапазоны объёма (SOFT, вторичные) ✅/🔧
- Типовой диапазон ~**140–200 мл/кг/сут** (AAP) / **150–200 мл/кг/сут** (NICE BNFC, первые 4 мес). 
- **Правка кросс-сверки:** держать это как **вторичный soft-ориентир (severity = `info`), не критический guardrail.** Якорные числа — AAP per-pound (165) и таблица FAO; диапазоны из snippet’ов используются только для мягкой проверки «не выходит ли расчёт за типичное».
- Мягкий суточный потолок ~960 мл (AAP, §3.2): severity = `warning`, source = `AAP`.
Источники: AAP; NICE BNFC *Enteral nutrition* — `https://bnfc.nice.org.uk/treatment-summaries/enteral-nutrition/`

### 7.3. ⚙️ Предупреждения research-уровня (отдельное ведро, НЕ guideline)
- Объём **≥ 840 мл/сут** ассоциирован с повышенным риском избыточной массы — данные **одного когортного исследования** (Huang et al., *Nutr J.* 2018), а не норматив. severity = `research_note`, помечать как research, отделять от AAP-960.
- *Примечание сверки:* допустимо вообще не включать этот флаг (одно исследование). Если включаете — только как research_note.

### 7.4. Частота кормлений по возрасту

| Возраст | Кормлений/сут |
|---|---|
| 0–1 нед | 8–12 |
| 1 нед – 1 мес | 8–10 |
| 1–2 мес | 6–8 |
| 2–4 мес | 5–7 |
| 4–6 мес | 4–6 |
Источник: AAP; CDC.

### 7.5. Single-feed sanity check — эвристика, НЕ клинический ориентир 🔧⚙️

Это НЕ guideline-потолок. Авторитетного in-scope стандарта «ёмкость желудка как
лимит объёма кормления» для детей 0–6 мес на искусственном вскармливании НЕ
существует: AAP/CDC дают типовые объёмы кормления (§7.6), а не валидируемый
stomach-cap. Источник цифры 30 мл/кг — PAHO/WHO Guiding Principles for
Complementary Feeding (2003), то есть руководство по ПРИКОРМУ для детей 6–24 мес,
которое к тому же отменено (заменено WHO Guideline 2023). Поэтому 30 мл/кг
приводится ТОЛЬКО как контекст функциональной ёмкости, не как порог.

Прежняя таблица «ёмкость по возрасту» (день 1: 5–20 … 1 мес: 80–150) УДАЛЕНА:
она происходит из лактационного образовательного конструкта «newborn stomach
size», описывает первую неделю и давала ложные флаги после ~2 недель (баг:
объём 100 мл у ребёнка 23 дней триггерил флаг по устаревшей точке «день 7 = 60 мл»).

Функциональная ёмкость (reference, НЕ порог): ~30 мл/кг на кормление
(PAHO/WHO 2003, контекст прикорма; superseded).

Проверка — весовая и зонированная по возрасту:

| Возраст     | Проверка объёма на 1 кормление                                   | severity |
|-------------|------------------------------------------------------------------|----------|
| 0–7 дней    | `per_feed_ml > 20 × weight_kg` → флаг                            | info     |
| 8–14 дней   | без cap; только per-feed guidance (§7.6) + responsive feeding    | —        |
| > 14 дней   | НЕ сравнивать с ёмкостью; `per_feed_ml > 40 × weight_kg` → флаг  | info     |

Назначение порогов:
- 20 мл/кг (0–7 дней): раннее перекармливание за раз связано с пищевой
  непереносимостью (Frontiers Pediatr 2023; >10,5 мл/кг в первые сутки —
  доказанный сигнал). Это эвристика осторожности, не клинический cap.
- 40 мл/кг (>14 дней): грубый отлов ОШИБОК ВВОДА (напр. «400 мл в одной
  бутылочке»), а не клинический сигнал. Порог с запасом перекрывает и
  функциональную ёмкость 30 мл/кг, и верхние значения типовых объёмов §7.6,
  поэтому НЕ срабатывает на нормальном кормлении.

ВАЖНО: §7.6 (типовые объёмы) с этой проверкой НЕ сравнивать — это нормальные
кормления по определению. Сравнение §7.6 ↔ §7.5 = категориальная ошибка.

Проверка на кейсе: 23 дня, 3,95 кг, 100 мл → зона >14 дней, 40 × 3,95 = 158 мл,
100 < 158 → флага нет. (Прежнее поведение: ложный флаг.)

### 7.6. Типовой объём на одно кормление (UI)

| Возраст | Объём/кормление |
|---|---|
| Первые дни | 30–60 мл |
| Конец 1 мес | 90–120 мл |
| 2 мес | 90–150 мл |
| 3–4 мес | 120–180 мл |

Источник: AAP; CDC.

## 8. Ветка недоношенных / низкой массы (preterm / LBW) — опционально ⚙️
Реализовывать только как отдельный clinician-mode, не как массовый калькулятор для родителей. Область применения первоисточника — недоношенные с массой при рождении < 1800 г.

### 8.1. Решение о типе смеси — WHO 2022
- GA < 32 нед **или** масса при рождении < 1,5 кг: preterm formula может рассматриваться, если нет молока матери и донорского молока.
- GA 32–36 нед **или** масса 1,5–2,4 кг: твёрдой рекомендации нет → clinical judgement.
Источник: WHO, *Recommendations for care of the preterm or low-birth-weight infant* (2022) — `https://iris.who.int/items/4a14b190-bf5a-430d-982a-8bcaf9ae3562`

### 8.2. Количественный расчёт — ESPGHAN 2022
- Энергия: **115–140 ккал/кг/сут**. При плохом росте допустимо > 140, но **не > 160**, и только после подтверждения достаточности белка/нутриентов.
- Объём: обычно **150–180 мл/кг/сут**; безопасный нижний предел **135 мл/кг/сут** (если потребности иначе закрыты); индивидуальный верхний — **до 200 мл/кг/сут**.
- Если плотность не даёт уложиться в 135–200 мл/кг — рекомендовать пересмотр плотности / clinician review, а не «просто больше мл».

🔧 **Точная ссылка (правка кросс-сверки):** Embleton ND, Moltu SJ, Lapillonne A, et al. *Enteral Nutrition in Preterm Infants (2022): A Position Paper From the ESPGHAN Committee on Nutrition and Invited Experts.* J Pediatr Gastroenterol Nutr. 2023;76(2):248–268. DOI: 10.1097/MPG.0000000000003642. PMID: 36705703.

### 8.3. Рефлюкс / срыгивания — NICE
Не снижать суточный объём автоматически. Если объём чрезмерен для веса — уменьшить; иначе сохранить общий суточный объём и разбить на меньшие, более частые кормления.
Источник: NICE, GORD in children and young people.

## 9. Что ИСКЛЮЧЕНО из движка (с обоснованием) ⛔

| Исключено | Причина |
|---|---|
| **EER-уравнение вида `−716,45 − … + height + weight`** | Использует возраст в **годах**, не месяцах, и относится к «6 мес – 2,99 года». При подстановке месяцев женское уравнение завышает калории на ~25–27%. Замена — FAO (§2) или IOM (§2.5). |
| **Формулы Финкельштейна, Апперта, дольный метод 1/5–1/7** | Исторические; не входят в современные стандарты. Только в справочной части, не в калькуляторе и не как дефолт. |
| **Holliday–Segar (100 мл/кг) и посуточная эскалация мл/кг (NICE NG29)** | Это **поддерживающая / инфузионная жидкость в стационаре**, а не стандарт кормления смесью. В движок не брать. Если показывать ramp по дням — только как «неонатальная общая потребность в жидкости (справочно)», не как целевой объём смеси. |
| **Жёсткий cap «AAP/CDC 960»** | CDC лимита не формулирует; потолок только AAP и мягкий. В коде — soft warning, source = `AAP`. |
| **840 мл как guideline-лимит** | Результат одного исследования; только `research_note` (или не включать). |

## 10. Опциональная развилка (EFSA/EU) ⚙️
ЕС считает, что современная смесь по составу приблизилась к грудному молоку, поэтому для искусственников применимо уравнение грудного вскармливания (более низкие значения). Значит, «formula-fed» оценка может слегка **завышать** для смесей нового поколения. Это **не настолько универсально**, чтобы заменять дефолт.
```
energy_basis = "FAO_formula_fed"  // ДЕФОЛТ (консервативно выше)
energy_basis = "EU_adjusted"      // опциональный режим
```
Источник: EFSA scientific opinions; обзор PMC8575726.

```text
INPUT: age_days, weight_kg, D_label_100ml?, D_label_oz?,
       standard_formula=true, mode="FAO_energy"|"AAP_simple"
# preterm — отдельный clinician-mode (§8)

# плотность
if D_label_100ml: D = D_label_100ml
elif D_label_oz:  D = D_label_oz / 29.5735 * 100
else:             D = 67; flag "estimated_density_default"

# валидация
assert weight_kg > 0 and age_days >= 0
if standard_formula and not (60 <= D <= 70): warn "density_out_of_codex_range"

# ранний период (product policy)
if age_days < EARLY_NEWBORN_DAYS:    # default 14, configurable
  return { type:"early_newborn_guidance", per_feed_ml:"30–60",
           feeds_per_day:"8–12", note:"single daily volume not provided" }

# основной расчёт
month   = resolve_month_bucket(age_days)         # 1..6, задокументировать округление
kcal_kg = FAO_FORMULA_FED[month]                 # {1:120,2:109,3:100,4:87,5:86,6:84}
V_fao   = weight_kg * kcal_kg * 100 / D

V_aap_raw = weight_kg * 165
V_aap     = min(V_aap_raw, 960)                   # soft cap, source=AAP

primary = (mode=="AAP_simple") ? V_aap : V_fao

# флаги (рекомендательный слой — на расчётных числах)
ml_per_kg = primary / weight_kg
if ml_per_kg < 140 or ml_per_kg > 200: flag {info: "outside typical AAP/NICE range"}
if V_aap_raw > 960:                    flag {warning: "AAP soft cap applied", source:"AAP"}
if primary >= 840:                     flag {research_note: "Huang 2018 association"}  # optional

# single-feed sanity check (эвристика, НЕ клинический cap) — см. §7.5
# functionalCapacityMl = 30 * weight_kg   # PAHO/WHO 2003, reference-only, superseded
# зона 0–7 дней — на РЕКОМЕНДОВАННОМ объёме (в раннем периоде нет суточной нормы):
recommended_per_feed = early_newborn ? 30 : (primary / feeds_per_day)
if age_days <= 7 and recommended_per_feed > 20 * weight_kg:
  flag {info: "large_single_feed_early_newborn", perFeedMl, weightKg}
# зона 8–14 дней — без cap (только per-feed guidance §7.6 + responsive feeding)

# зона > 14 дней — В СЛОЕ ФАКТИЧЕСКИХ КОРМЛЕНИЙ, не в рекомендательном расчёте.
# input: реальные feedings за день + вес; берётся МАКСИМАЛЬНОЕ одиночное кормление
# (не среднее), цель — отлов ошибок ввода («400 мл в одной бутылочке»):
#   if age_days > 14 and max_single_feed_ml > 40 * weight_kg:
#     flag {info: "single_feed_unusually_large_for_weight", perFeedMl, weightKg}

return {
  primary_ml_day: round(primary),
  method: mode,
  sanity_check_ml_day: round((mode=="AAP_simple") ? V_fao : V_aap),
  density_used: D,
  flags: [...],
  disclaimer: "Расчёт — ориентир. Responsive feeding: кормление по сигналам голода/сытости. При сомнениях — педиатр."
}
```

## 14. Оговорки
- Документ — для **здоровых доношенных** детей 0–6 мес на полном искусственном вскармливании. Недоношенные, ЗВУР, лечебные смеси, пороки сердца/метаболические болезни — вне базового режима (см. §8 и индивидуальный расчёт).
- Любой расчётный объём — **ориентир**, не цель потребления. Базовый принцип (WHO, AAP, CDC) — responsive feeding.
- Приложение не заменяет педиатра. При отклонениях роста, недоборе/избытке массы, систематическом выходе за ориентировочные диапазоны — направлять к врачу и оценке по WHO growth charts / weight-for-length / BMI-for-age.
- Это не медицинская рекомендация и не юридический документ.
